# ハンズオンの練習

自分の作業用のフォルダをパソコン上に用意してください。

```
/works
```

`works` ディレクトリの中に `test.txt` というファイルを作ってください。
この資料では今の `works` ディレクトリの中を次のように表現します。

```
/works
└── test.txt
```
