# AIお絵描きハンズオン

このハンズオンでは、HTML、CSS、JavaScriptを使ったWebアプリケーション開発の基礎から、Canvas APIを使ったお絵描きアプリの実装まで、段階的に学習します。

## 対象者

- プログラミング初心者の方
- Web開発の基礎を学びたい方
- Canvas APIを使った描画を体験したい方

## 必要な環境

- テキストエディタ（VS Code、メモ帳など）
- Webブラウザ（Chrome、Firefox、Safariなど）

## ハンズオンの進め方

このハンズオンは3つの章で構成されています。**第1章から順番に進めてください。**

### 第1章: Webフロントエンドの基本

[📘 第1章を開く](./docs/1-basic/README.md)

HTML、CSS、JavaScriptの基本的な書き方を学びます。

1. [Hello World](./docs/1-basic/1-hello-world/README.md) - HTMLの基本
2. [装飾](./docs/1-basic/2-styling/README.md) - CSSでの見た目の調整
3. [インタラクション](./docs/1-basic/3-interaction/README.md) - JavaScriptでの動的な操作

### 第2章: お絵描きアプリの実装

[📘 第2章を開く](./docs/2-drawing/README.md)

Canvas APIを使って、お絵描きアプリケーションを段階的に作成します。

1. [Canvasの基本的な使い方](./docs/2-drawing/1-canvas/README.md) - Canvas APIの基礎
2. [マウスで線を描く](./docs/2-drawing/2-draw-line/README.md) - マウス操作による描画
3. [画像をキャンバスに描く](./docs/2-drawing/3-put-image/README.md) - 画像の読み込みと描画

### 第3章: スコアリング（準備中）

※ 現在準備中です

## 作業フォルダの準備

ハンズオンを始める前に、作業用のフォルダをパソコン上に用意してください。

```
/works
```

各章の演習では、この `works` フォルダの中にファイルを作成していきます。

## 画像素材について

第2章のお絵描きアプリで使用する画像ファイルは、各セクションの `examples` フォルダに用意されています。必要に応じて参照してください。

## 学習のポイント

- **段階的な学習**: 各セクションは前のセクションの知識を基に構成されています。必ず第1章から順番に進めてください。
- **実際に手を動かす**: コードを読むだけでなく、実際に自分で書いて動かしてみることが重要です。
- **動作確認**: 各ステップでブラウザで動作を確認しながら進めましょう。
- **エラーに慣れる**: エラーが出たら、エラーメッセージを読んで原因を探る練習をしましょう。
